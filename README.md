## 📖 Overview

This project is a self-practice endeavor to recreate the user interface of YouTube using fundamental web technologies: HTML, CSS, and JavaScript. It focuses on meticulously replicating the visual layout, styling of the popular video-sharing platform's frontend. Designed as a learning exercise, it demonstrates proficiency in structuring web pages, applying modern CSS techniques, and implementing basic UI interactivity.

## ✨ Features

-   🎯 **YouTube-like Layout**: Faithful recreation of the YouTube homepage layout, including header, sidebar, and video content grid.
-   📱 **Responsive Design**: Adapts seamlessly across various screen sizes (desktops, tablets, mobile) for an optimal viewing experience.
-   🔍 **Interactive Search Bar**: Visually implemented search input field in the header.
-   📺 **Dynamic Video Grid**: Displays a grid of video thumbnail cards with titles, channel names, and view counts.
-   ➡️ **Collapsible Sidebar**: Basic JavaScript functionality to toggle the visibility of the navigation sidebar.

## 🚀 Quick Start

This project is a static HTML/CSS/JS application and does not require any build tools or development servers.

### Prerequisites
-   A modern web browser (e.g., Chrome, Firefox, Safari, Edge).

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Sehar-1207/Youtube-Clone.git
    cd Youtube-Clone
    ```

2.  **Open in your browser**
    Simply open the `youtube.html` file in your preferred web browser.
    ```bash
    # On most systems, you can simply run:
    open youtube.html
    # Or navigate to the file in your file explorer and double-click it.
    ```
